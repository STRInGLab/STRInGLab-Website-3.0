# S.T.R.In.G Design Handoff

This directory is the production brief for S.T.R.In.G Technology Solutions. It translates the existing website's visual system into rules a design team can use to create campaign images, social assets, sales collateral, decks and templates.

## Start here

- [Design system](design-system.md) - colours, type, layout, components and accessibility rules.
- [Design language and art direction](design-language.md) - the visual point of view, image direction and do/don't guidance.
- [Image and template production kit](image-template-kit.md) - ready-to-use prompts, crops and template specs.
- [Presentation template guide](presentation-template-guide.md) - master-slide set and deck rules.
- [Presentation template PDF](pdf/stringlab-presentation-template.pdf) - visual reference for the slide system.
- [Design tokens](tokens.json) - machine-readable starter tokens for Figma, design tooling and implementation.

## Source of truth

The current website styles are the implementation reference: `assets/shared.css`. The existing wordmark source is `assets/string-logo.png`; retain its proportions and do not redraw, stretch or add effects to it.

## Required first delivery from the design team

1. A shared Figma library with the colour, type and layout tokens below.
2. A master set of seven presentation layouts described in the guide.
3. Image templates in the requested aspect ratios, plus 6-10 launch visuals using the prompt kit.
4. Export presets for web, social, email and presentation use.

All copy, claims, logos and client marks need a final internal approval before publication.
