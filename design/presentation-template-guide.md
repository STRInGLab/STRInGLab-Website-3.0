# Presentation Template Guide

## Build specification

Create a 16:9 PowerPoint and Google Slides master template at 1920 x 1080 px. The working grid has 12 columns, 120 px outer margins, 24 px gutters and a 72 px footer zone. Create masters with both dark (ink) and light (paper) variants; use light slides sparingly as a reading reset.

Use Newsreader for titles, Inter Tight for body copy and JetBrains Mono for metadata. Embed or package the licensed fonts where the platform permits; otherwise use the documented fallbacks.

### Required slide masters

| # | Layout | Content rule |
| --- | --- | --- |
| 01 | Cover | One title, a short descriptor and optional date/client. No busy visual. |
| 02 | Section divider | Large section number or label, 2-6 word title, one green accent. |
| 03 | Point of view | One assertion, brief supporting copy and one editorial image. |
| 04 | System / process | 3-5 stages on a logical route; highlight one active or recommended stage. |
| 05 | Case-study outcome | One large metric, concise context and one proof image or chart. |
| 06 | Evidence / data | One chart or table, direct conclusion and a mono source line. |
| 07 | Closing / CTA | Final statement, contact details and restrained wordmark. |

## Slide rules

- One message per slide. If two assertions need defending, use two slides.
- Use a 52-72 pt title on a cover; 36-44 pt for slide titles; 18-22 pt for body copy.
- Place a mono footer: `S.T.R.In.G / CONFIDENTIAL` on client work and slide number aligned right.
- Keep each slide to one hero image, one chart or one system illustration - not all three.
- Use cards only when comparing repeated objects. Prefer open canvas and well-spaced text for narrative slides.
- Charts use paper fields or a minimally bordered ink field; direct-label important series.
- Do not add template logos to every slide. A small wordmark on cover and close is enough; the footer can use a text-only signature.

## Recommended deck arc

1. Cover
2. The business context
3. The operational problem
4. The insight or opportunity
5. Proposed system / approach
6. Evidence or product proof
7. Expected impact and delivery path
8. Closing / next step
